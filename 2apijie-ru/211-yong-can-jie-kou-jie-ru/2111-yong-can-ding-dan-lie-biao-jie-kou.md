2.11.1 用餐订单列表

####接口说明
- 1.**最新接口**
- 2.查询用餐订单列表接口
  通过该接口可以查询企业使用分贝通APP下的用餐订单信息






请求方式|请求地址
----|---
POST|/open/api/dinner/order/list


字段|名称|类型|必填|描述
-----|-----|----|----|----
timestamp|时间戳 |long |Y|13位时间戳
sign|签名 |string |Y|lw8osdfj983uh4
access_token|token | string |Y|登录 token
employee_id| 操作人id|string |Y|操作人id,调用接口人 id
employee_type| 用户类型|string|Y|类型，0为分贝用户，1为第三方用户
data |请求数据| jsonobject |Y|请求数据
data.state| 查询状态|integer |Y|0全部
data.search\_category |搜索维度| integer | N |1:企业，2:个人
data.page\_index |页码| integer | N |1(默认值)
data.page\_size|每页显示条数| integer| N |20(默认值)


请求示例：

```



"access_token":"5747fbc10f0e60e0709d8d722",
"sign":"oihfnlyeofdh98",
"timestamp":124124325,
"employee_id":"59b74c1323445f2d54dd07922",
"employee_type":"1",
"data":{
"state":0
}



```
